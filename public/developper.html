Développer

Dans ce chapitre, tu vas apprendre à développer des expressions algébriques en utilisant la distributivité et les identités remarquables.

Cours Exercices Vidéo Astuce

Cours

1. Distributivité simple

a(b + c) = ab + ac

Exemple : 3(x + 2) = 3x + 6

2. Double distributivité

(a + b)(c + d) = ac + ad + bc + bd

Exemple : (x + 2)(x + 3) = x² + 5x + 6

3. Identités remarquables

  • (a + b)² = a² + 2ab + b²
  • (a - b)² = a² - 2ab + b²
  • (a - b)(a + b) = a² - b²

Exemple

Développer : (x + 4)²

= x² + 8x + 16

Exercices

Exercice 1

Développer :

  • 2(x + 5)
  • 3(x - 4)
  • -2(x + 3)

Exercice 2

Développer :

  • (x + 2)(x + 3)
  • (x - 1)(x + 4)

Exercice 3

Utiliser les identités remarquables :

  • (x + 5)²
  • (x - 3)²
  • (x - 2)(x + 2)

Exercice 4

Développer et réduire :

  • (2x + 3)(x + 5)
  • (x + 1)(x + 1)

Astuce

Pense à distribuer chaque terme : un nombre devant une parenthèse multiplie tous les termes à l’intérieur.

Pour les identités remarquables, essaie de les reconnaître directement : cela te fait gagner beaucoup de temps.

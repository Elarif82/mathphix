Puissances

Dans ce chapitre, tu vas apprendre à manipuler les puissances, utiliser les règles de calcul et simplifier des expressions.

Cours Exercices Vidéo Astuce

Cours

1. Définition

Une puissance est une écriture répétée d’une multiplication.

aⁿ = a × a × ... × a (n fois)

Exemple : 2³ = 2 × 2 × 2 = 8

2. Règles de calcul

  • aᵐ × aⁿ = aᵐ⁺ⁿ
  • aᵐ / aⁿ = aᵐ⁻ⁿ (a ≠ 0)
  • (aᵐ)ⁿ = aᵐⁿ
  • (ab)ⁿ = aⁿ bⁿ

3. Puissance de 10

10ⁿ permet d’écrire les grands nombres.

Exemple : 1000 = 10³

4. Exposant négatif

a⁻ⁿ = 1 / aⁿ

Exemple : 2⁻³ = 1 / 8

Exemple

Simplifier : 2³ × 2²

= 2⁵ = 32

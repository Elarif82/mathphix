Factoriser

Dans ce chapitre, tu vas apprendre à transformer une somme en produit en utilisant la mise en facteur et les identités remarquables.

Cours Exercices Vidéo Astuce

Cours

1. Mise en facteur

Factoriser consiste à écrire une expression sous forme de produit.

ab + ac = a(b + c)

Exemple : 3x + 6 = 3(x + 2)

2. Mise en facteur avec un signe négatif

On peut factoriser par un nombre négatif.

Exemple : -2x - 4 = -2(x + 2)

3. Identités remarquables

  • a² + 2ab + b² = (a + b)²
  • a² - 2ab + b² = (a - b)²
  • a² - b² = (a - b)(a + b)

4. Factorisation par regroupement

Exemple :

ax + ay + bx + by = a(x + y) + b(x + y)

= (a + b)(x + y)

Exemple

Factoriser : x² + 6x + 9

= (x + 3)²

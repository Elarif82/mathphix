Cours

Les notions essentielles du calcul littéral

Des rappels simples pour poser les bonnes bases.

1

Comprendre une expression

Une expression comme 3x + 5 contient un nombre, une lettre et des opérations. La lettre représente une valeur inconnue ou variable.

2

Réduire

Réduire, c’est regrouper les termes de même nature. Par exemple : 3x + 5x = 8x.

3

Développer

Développer, c’est enlever les parenthèses. Exemple : 3(x + 4) = 3x + 12.

4

Factoriser

Factoriser, c’est mettre un facteur commun en évidence. Exemple : 4x + 8 = 4(x + 2).

Méthode

Comment progresser en calcul littéral ?

Une méthode simple pour éviter de se perdre.

1. Repérer les termes semblables.
2. Réduire seulement ce qui peut l’être.
3. Développer avec soin en distribuant correctement.
4. Vérifier le signe des termes à chaque étape.

Erreurs fréquentes

À éviter absolument

Erreur 1 : écrire 3x + 2 = 5x.

On ne peut pas additionner un terme en x avec un nombre seul.

Erreur 2 : mal distribuer dans une parenthèse.

Exemple faux : 2(x + 3) = 2x + 3.

Erreur 3 : oublier un signe moins devant une parenthèse.

Cours

1. Définition

Un intervalle est un ensemble de nombres compris entre deux bornes.

2. Types d’intervalles

  • [2 ; 5] → bornes incluses
  • ]2 ; 5[ → bornes exclues
  • [2 ; 5[ → demi-ouvert
  • [0 ; +∞[

3. Réunion et intersection

A ∪ B : on rassemble

A ∩ B : partie commune

A = [1 ; 4], B = [3 ; 6]

A ∩ B = [3 ; 4]

A ∪ B = [1 ; 6]

Exercices

Décris :

  • [2 ; 7]
  • ]-1 ; 4]

Soient A = [0 ; 5], B = [3 ; 8]

  • A ∩ B
  • A ∪ B
